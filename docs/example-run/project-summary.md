# Project Summary — Twenty

## Compiled truth — what is true NOW

- **Offering:** Open-source CRM platform for technical teams; extensible via custom objects, server-side logic, UI components, and AI agents — all as TypeScript packages. Positions as "build your enterprise CRM at AI speed." — [source](https://twenty.com)
- **Users / customers:** Large enterprises replacing Salesforce, startups with technical founders, GTM teams wanting custom lead scoring/enrichment, privacy-conscious orgs needing self-hosting, Salesforce partners seeking better DX, and web dev agencies familiar with TypeScript/React/PostgreSQL. — [source](https://docs.twenty.com/getting-started/introduction)
- **Business model:** Self-hosted open-source core (AGPL); cloud-hosted Pro tier at $9/user/month and Organization tier at $19/user/month (yearly -25%). Revenue from cloud hosting, priority support, SSO/SAML, row-level permissions, custom domains, and implementation partner network. — [source](https://twenty.com/pricing)
- **Tech stack:** React + TypeScript frontend, Node.js backend, PostgreSQL database. GraphQL and REST APIs, webhooks, OAuth, MCP server. Workflows engine with triggers, conditions, HTTP requests, serverless functions. Version control via git-backed workspaces (as of v2.0). — [source](https://docs.twenty.com/developers/introduction)
- **Differentiators:** (1) Open-source, self-hostable, no vendor lock-in; (2) AI-native customization — Cursor/Claude Code/CLI can scaffold entire apps on the platform; (3) TypeScript-only DX vs. Salesforce's proprietary Apex; (4) MCP server exposing workspace to any MCP client; (5) 90%+ cost reduction vs. Salesforce for self-hosters. — [source](https://twenty.com/why-twenty)
- **Stated strategy:** "The future of CRM is built, not bought." Differentiation lives in code you own; AI made the gap small enough to build custom CRM in an afternoon. Targeting the shift from CRM-as-ledger to CRM-as-operating-system via AI agents. — [source](https://twenty.com/why-twenty)
- **Constraints:** Still growing integration ecosystem (not yet as broad as Salesforce/HubSpot); rewards teams that want to stay close to tools; no pre-built enterprise marketplace yet; free-tier API rate limit 50 req/min. — [source](https://docs.twenty.com/getting-started/introduction)
