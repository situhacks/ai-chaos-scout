# Deck example — Twenty (illustrative)

Illustrative demo of the Instar deck-template applied to a sample `report.json` for
**Twenty** (open-source CRM, `github.com/twentyhq/twenty`, `twenty.com`).

These SVGs are hand-rendered from the template layouts to prove the design language.
They are **not** generated by the ppt-master pipeline — they show what a final deck
slide looks like when the Instar tokens are applied.

## Files

| File | Slide type |
|---|---|
| `report.json` | Sample ReportModel for Twenty (5 recs, 2 themes) |
| `cover-twenty.svg` | Cover slide with metamorphosis strip |
| `track-a1-twenty.svg` | Track A specimen plate: "AI Agent SDK for CRM Workflows" |
| `track-b1-twenty.svg` | Track B metamorphosis card: "AI-Native CRM Platform Pivot" |

Open any `.svg` file in a browser to preview. The slides use 1280x720 viewBox (PPT 16:9).

## Design language visible in these slides

- Warm paper ground (`#f4efe4`)
- Specimen-blue ink for Track A (`#1f3a5f`)
- Madder-red for Track B (`#8c3b2e`) with margin warning stripe
- Graph-paper grid (20px pitch, 6% opacity)
- Hairline-ruled specimen-plate cards
- 5-cell slider strips in track color
- Monospace citation codes and specimen codes
- Metamorphosis-stage glyphs (larva for A1/L2, emergence for B1/L4)
- Kill-criteria callout box on Track B
- Georgia serif titles, Arial sans body, Consolas monospace codes
