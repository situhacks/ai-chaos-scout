Deep Dive — Digest Themes
LLM Integration Wave Hits Enterprise Analytics
Three major analytics platforms shipped natural-language query features this quarter. Databricks Genie went GA with SQL generation, Snowflake Cortex launched NL-to-SQL for all Enterprise accounts, and Microsoft Copilot Analytics entered public preview. The convergence suggests NL analytics is becoming table-stakes rather than a differentiator.
Sources: Databricks Genie GA, Snowflake Cortex, Microsoft Copilot Analytics
Open Source Analytics Gains Momentum
Grafana Labs hit $12B valuation proving the OSS analytics model works at scale. Metabase 1.0 shipped its embedded analytics SDK, making it trivial to white-label analytics into other products. Meanwhile, HashiCorp&#x27;s BSL retrospective showed that open-core companies retain developer trust better than proprietary-first competitors.
Sources: Grafana $12B, Metabase 1.0, HashiCorp BSL retrospective
Real-Time Data Processing Goes Mainstream
Confluent launched Tableflow bridging streaming and analytics workloads in a single product. Grafana added streaming SQL to its query engine, and the Y Combinator S26 batch featured several real-time analytics startups. The boundary between operational and analytical data continues to blur.
Sources: Confluent Tableflow, Grafana streaming SQL, YC S26 batch
